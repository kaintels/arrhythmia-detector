## 1. 개요

- 2015년 기준, 유엔의 보고서에 따르면 세계는 고령화 시대로 접어들고 있음
- 또한 사람은 보통 나이가 들 수록 심혈관 시스템이 약해지며 부정맥과 같은 심혈관 질환에 취약해진다고 알려져 있음

- 본 프로젝트는 생체신호의 원시 신호만을 이용하고 딥러닝을 활용하여 부정맥 자동감지를 수행
